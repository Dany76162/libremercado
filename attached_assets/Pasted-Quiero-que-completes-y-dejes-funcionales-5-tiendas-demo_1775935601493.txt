Quiero que completes y dejes funcionales **5 tiendas demo** dentro del sistema de LibreMercado, usando los **5 reels ya subidos** como base visual y comercial de cada tienda.

Dejar armadas **5 tiendas completas y realistas**, listas para mostrarse en el marketplace, en el panel y en la home, como si ya estuvieran preparadas para salir en demo o presentación.

## Lo que tenés que hacer

### 1. Crear y completar 5 tiendas demo

Cada una debe quedar con:

* nombre de tienda
* logo o imagen principal
* banner o portada
* descripción comercial
* categoría
* ubicación
* reputación o rating simulado
* cantidad de ventas simuladas
* estado activo
* datos completos visibles en el panel del vendedor y en la vista pública

Las tiendas tienen que verse reales, atractivas y distintas entre sí.

---

### 2. Asociar un reel a cada tienda

Ya hay **5 reels subidos de 5 tiendas**.
Quiero que cada tienda tenga su reel correspondiente correctamente vinculado.

Cada reel debe:

* aparecer en la tienda
* aparecer en la sección de videos / ReelMark
* quedar asociado a un producto principal de esa tienda
* verse también en el panel del vendedor si corresponde
* tener miniatura si el sistema la usa
* tener título, descripción corta y métricas simuladas (views, likes, etc.) si ya existe esa estructura

---

### 3. Generar productos para cada tienda

Cada tienda debe tener productos completos y coherentes con el reel.

Para cada tienda:

* crear varios productos
* al menos 1 producto principal vinculado al reel
* al menos 3 productos visibles por tienda
* cada producto debe tener:

  * nombre
  * descripción
  * precio actual
  * precio anterior si aplica
  * stock
  * categoría
  * imágenes
  * producto destacado si corresponde
  * tienda asociada
  * estado activo

Los productos deben estar bien presentados y listos para mostrarse en:

* tienda pública
* home
* destacados
* panel del vendedor

---

### 4. Cargar imágenes relacionadas a cada tienda y producto

Quiero que el sistema use imágenes **similares visualmente al contenido del reel** de cada tienda.

Necesito que:

* cada tienda tenga **3 imágenes**
* esas imágenes sean coherentes con el estilo o producto mostrado en el reel
* se usen como galería del producto o de la tienda
* si no hay imágenes disponibles en base de datos, generar seed/demo assets o mock data visual coherente
* si ya existe soporte de imágenes múltiples, cargar las 3 correctamente
* si no existe, implementarlo para que cada producto o tienda pueda mostrar 3 imágenes

La idea es que cuando entre a cada tienda se vea:

* reel
* producto principal
* galería de imágenes
* precio
* descripción
* información completa
